# Lab: Bài Tập Thực Hành Tổng Hợp

## Giới thiệu
Thư mục Lab chứa các bài tập thực hành tổng hợp nhằm củng cố kiến thức đã học từ các bài lý thuyết. Các bài tập được phân loại từ mức độ cơ bản (Hello World) đến các bài toán xử lý logic phức tạp hơn.

## Danh sách bài thực hành

### Lab 01: Hello World
- Mục đích: Làm quen với cú pháp cơ bản và kiểm tra cấu hình môi trường.
- Nội dung: Sử dụng `print()` để in chuỗi ra màn hình.

### Lab 02: Bài Tập Tổng Hợp
- Mục đích: Áp dụng biến, kiểu dữ liệu và toán tử.
- Nội dung: Các bài toán tính toán cơ bản, xử lý chuỗi và số học.

### Lab 03: Logic và Thuật Toán
- Mục đích: Rèn luyện tư duy logic với cấu trúc điều kiện và vòng lặp.
- Nội dung: Xử lý các bài toán kiểm tra số, tìm kiếm trong danh sách.

### Lab 04 - 06: Nâng Cao
- Mục đích: Làm quen với hàm, xử lý lỗi và các cấu trúc dữ liệu nâng cao.
- Nội dung: Xây dựng các hàm xử lý dữ liệu người dùng, quản lý trạng thái đơn giản.

## Hướng dẫn thực hiện
1. Đọc kỹ yêu cầu đề bài được ghi chú trong các file `.dart`.
2. Mở file trong VS Code và thực hiện viết mã nguồn.
3. Chạy chương trình bằng terminal: `dart run [tên_file].dart`.
4. Kiểm tra kết quả output so với yêu cầu.

## Quy tắc viết code
- Sử dụng tên biến có nghĩa (meaningful names).
- Căn lề code sạch sẽ, rõ ràng.
- Sử dụng comment để giải thích những đoạn code phức tạp.
- Tuân thủ quy tắc Dart Style (CamelCase cho biến và hàm).
